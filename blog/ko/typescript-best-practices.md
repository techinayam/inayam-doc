## 1. 엄격한 모드를 사용하십시오

TypeScript의 엄격한 모드를 사용하면 언어를 더 안전하고 예측 가능하게 만드는 몇 가지 컴파일러 옵션이 가능합니다.엄격한 모드를 활성화하면 유형에 명시 적으로 명시되어있어 코드의 잠재적 오류를 방지합니다.

엄격한 모드를 활성화하려면`tsconfig.json`에 다음을 추가하십시오.

```JSON
{
"컴파일러 옵션": {
"엄격한": 사실
}
}
