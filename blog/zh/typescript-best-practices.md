## 1。使用严格模式

打字稿的严格模式可实现多种编译器选项，以使语言更安全，更可预测。使严格的模式迫使您使用类型明确表明，从而防止代码中的潜在错误。

要启用严格的模式，请将以下内容添加到您的“ tsconfig.json”：

``JSON
{
“ compileroptions”：{
“严格”：是的
}
}
