###例の計算
mol/m³の使用を説明するために、2立方メートルの水に溶解した0.5モルの塩化ナトリウム（NaCl）を含む溶液を検討してください。濃度は次のように計算できます。

\ [
\ text {濃度（mol/m³）} = \ frac {\ text {molesの数}} {\ text {volume（m³）}}} = \ frac {0.5 \ text {mol}} {2 \ text {m³}} = 0.25 \ text {mol/m³}}
\]
