## 초당 센티미터 (CM/S) 도구 설명

### 정의
초당 센티미터 (cm/s)는 1 초에 걸쳐 센티미터로 이동하는 거리를 측정하는 속도 단위입니다.이 메트릭은 일반적으로 물리 및 공학을 포함한 다양한 과학 분야에서 초당 미터 (m/s)보다 더 세분화 된 방식으로 물체의 속도를 표현하는 데 사용됩니다.

### 표준화
초당 센티미터는 메트릭 시스템의 일부로 국제적으로 인정 된 측정 시스템입니다.길이의 기본 단위 인 미터에서 파생되며, 여기서 1cm는 0.01 미터입니다.이 표준화는 다양한 과학 및 엔지니어링 응용 프로그램에서 일관성과 정확성을 보장합니다.

### 역사와 진화
속도를 측정하는 개념은 물리학의 초기 연구 연구로 거슬러 올라갑니다.초당 센티미터는 18 세기 후반 프랑스에서 설립 된 메트릭 시스템과 함께 진화했습니다.시간이 지남에 따라 CM/S는 작은 속도를 표현하기위한 편의로 인해 많은 과학 분야에서 선호되는 단위가되었습니다.

### 예제 계산
시간당 킬로미터 (km/h)를 초당 센티미터 (cm/s)로 변환하는 방법을 설명하려면 90km/h로 이동하는 차량을 고려하십시오.변환은 다음과 같이 수행 할 수 있습니다.

1. km/h를 m/s로 변환합니다.
\ [
90 \ text {km/h} \ times \ frac {1000 \ text {m}} {1 \ text {km}} \ times \ frac {1 \ text {h}} {3600 \ text {s}} = 25 \ text {m/s}
\]

2. m/s를 cm/s로 변환합니다.
\ [
25 \ text {m/s} \ times \ frac {100 \ text {cm}} {1 \ text {m}} = 2500 \ text {cm/s}
\]

따라서 90km/h는 2500cm/s에 해당합니다.
