## сантиметр в секунду (см/с) Описание инструмента

### Определение
Мединг в секунду (см/с) представляет собой единицу скорости, которая измеряет расстояние, пройденное в сантиметрах в течение одной секунды.Этот показатель обычно используется в различных научных областях, включая физику и инженерию, для выражения скорости объектов более гранулированным, чем метры в секунду (M/S).

### Стандартизация
Мединг в секунду является частью метрической системы, которая является всемирно признанной системой измерения.Он получен из базовой единицы длины, измерителя, где 1 см равняется 0,01 метра.Эта стандартизация обеспечивает согласованность и точность в различных научных и инженерных приложениях.

### История и эволюция
Концепция измерения скорости восходит к ранним исследованиям движения по физике.Мединг в секунду развивался вместе с метрической системой, которая была основана во Франции в конце 18 -го века.Со временем CM/S стали предпочтительным подразделением во многих научных дисциплинах из -за его удобства для выражения небольших скоростей.

### Пример расчет
Чтобы проиллюстрировать, как преобразовать километры в час (км/ч) в сантиметры в секунду (см/с), рассмотрим транспортное средство, проходящее в 90 км/ч.Преобразование может быть сделано следующим образом:

1. Преобразовать км/ч в М/с:
\ [
90 \ text {km/h} \ times \ frac {1000 \ text {m}} {1 \ text {km}} \ times \ frac {1 \ text {h}} {3600 \ text {s}} = 25 \ text {m/s}
\]

2. Преобразовать M/S в CM/S:
\ [
25 \ text {m/s} \ times \ frac {100 \ text {cm}} {1 \ text {m}} = 2500 \ text {cm/s}
\]

Таким образом, 90 км/ч эквивалентно 2500 см/с.
