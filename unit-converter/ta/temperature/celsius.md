## செல்சியஸ் வெப்பநிலை மாற்றி கருவி

### வரையறை
செல்சியஸ் (° C) என்பது வெப்ப ஆற்றலை அளவிட பயன்படுத்தப்படும் வெப்பநிலை அளவுகோலாகும்.இது உலகில், குறிப்பாக அறிவியல் சூழல்களிலும், அமெரிக்காவிற்கு வெளியே உள்ள பெரும்பாலான நாடுகளிலும் மிகவும் பரவலாகப் பயன்படுத்தப்படும் வெப்பநிலை அளவீடுகளில் ஒன்றாகும்.செல்சியஸ் அளவுகோல் 0 ° C வெப்பநிலையில் உறைபனி புள்ளியையும், நிலையான வளிமண்டல நிலைமைகளின் கீழ் 100 ° C க்கு கொதிநிலையையும் அடிப்படையாகக் கொண்டது.

### தரப்படுத்தல்
செல்சியஸ் அளவுகோல் சர்வதேச அலகுகளின் (எஸ்ஐ) தரப்படுத்தப்பட்டுள்ளது மற்றும் இது கெல்வின் அளவால் வரையறுக்கப்படுகிறது, அங்கு 0 ° C 273.15 K க்கு சமம். இந்த உறவு செல்சியஸுக்கும் கெல்வினுக்கும் இடையில் எளிதாக மாற்ற அனுமதிக்கிறது, இது அறிவியல் கணக்கீடுகள் மற்றும் பயன்பாடுகளுக்கு அவசியமாக்குகிறது.

### வரலாறு மற்றும் பரிணாமம்
செல்சியஸ் அளவுகோல் 1742 ஆம் ஆண்டில் ஸ்வீடிஷ் வானியலாளர் ஆண்டர்ஸ் செல்சியஸால் உருவாக்கப்பட்டது. முதலில், இது 100 டிகிரியில் தண்ணீரின் உறைபனி புள்ளியுடனும், கொதிநிலை புள்ளியிலும் 0 டிகிரியில் வரையறுக்கப்பட்டது.இருப்பினும், இது இன்று நாம் பயன்படுத்தும் தற்போதைய வரையறைக்கு மாற்றப்பட்டது.பல ஆண்டுகளாக, செல்சியஸ் அளவு பெரும்பாலான நாடுகளில், குறிப்பாக ஐரோப்பா மற்றும் ஆசியாவில் வெப்பநிலை அளவீட்டுக்கான தரமாக மாறியுள்ளது.

### எடுத்துக்காட்டு கணக்கீடு
ஃபாரன்ஹீட்டிலிருந்து (° F) இருந்து செல்சியஸ் (° C) ஆக வெப்பநிலையை மாற்ற, நீங்கள் சூத்திரத்தைப் பயன்படுத்தலாம்:
\ [
° C = (° F - 32) \ மடங்கு \ frac {5} {9}
\]
எடுத்துக்காட்டாக, 68 ° F ஐ செல்சியஸாக மாற்ற:
\ [
° C = (68 - 32) \ மடங்கு \ frac {5} {9} = 20 ° C.
\]

### அலகுகளின் பயன்பாடு
வானிலை, சமையல் மற்றும் அறிவியல் ஆராய்ச்சி உள்ளிட்ட பல்வேறு துறைகளில் செல்சியஸ் பரவலாகப் பயன்படுத்தப்படுகிறது.வானிலை முன்னறிவிப்புகள், சமையல் வெப்பநிலை மற்றும் ஆய்வக சோதனைகளைப் புரிந்துகொள்வதற்கு இது அவசியம்.

### பயன்பாட்டு வழிகாட்டி
செல்சியஸ் வெப்பநிலை மாற்றி கருவியுடன் தொடர்பு கொள்ள, இந்த படிகளைப் பின்பற்றவும்:
1. ** உள்ளீட்டு வெப்பநிலை **: நியமிக்கப்பட்ட உள்ளீட்டு புலத்தில் மாற்ற விரும்பும் வெப்பநிலையை உள்ளிடவும்.
2. ** மாற்று வகையைத் தேர்ந்தெடுக்கவும் **: நீங்கள் செல்சியஸிலிருந்து பாரன்ஹீட், கெல்வின் அல்லது நேர்மாறாக மாற்ற விரும்புகிறீர்களா என்பதைத் தேர்வுசெய்க.
3. ** மாற்றத்தைக் கிளிக் செய்க **: முடிவுகளை உடனடியாகக் காண 'மாற்ற' பொத்தானை அழுத்தவும்.
4. ** மதிப்பாய்வு முடிவுகள் **: மாற்றப்பட்ட வெப்பநிலை தெளிவாகக் காட்டப்படும், இது தேவைக்கேற்ப அதைப் பயன்படுத்த அனுமதிக்கிறது.

### அடிக்கடி கேட்கப்படும் கேள்விகள் (கேள்விகள்)

1. ** செல்சியஸை பாரன்ஹீட்டாக மாற்றுவதற்கான சூத்திரம் என்ன? **
- சூத்திரம் \ (° f = (° C \ முறை \ frac {9} {5}) + 32 \).

2. ** நான் செல்சியஸை கெல்வினுக்கு மாற்றுவது எப்படி? **
- செல்சியஸை கெல்வினாக மாற்ற, \ (k = ° C + 273.15 \) சூத்திரத்தைப் பயன்படுத்தவும்.

3. ** செல்சியஸில் தண்ணீரின் உறைபனி என்ன? **
- நீரின் உறைபனி புள்ளி 0 ° C ஆகும்.

4. ** நான் செல்சியஸை மற்ற வெப்பநிலை அலகுகளாக மாற்றுவது எப்படி? **

5. ** உலகளவில் செல்சியஸ் பயன்படுத்தப்பட்டதா? **
- ஆமாம், செல்சியஸ் என்பது உலகெங்கிலும் உள்ள பெரும்பாலான நாடுகளில் பயன்படுத்தப்படும் நிலையான வெப்பநிலை அளவுகோலாகும், இது அமெரிக்காவைத் தவிர, முதன்மையாக பாரன்ஹீட்டைப் பயன்படுத்துகிறது.
